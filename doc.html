TRIP REPORTING SERVICE

Get Trips

GET: http(s)://base-url/trip/getTrips
header: Authorization Bearer token

Optional Query Parameters

trip_id: (String) The id of a trip
driver_auth_id: (String) The auth Id of a driver
rider_auth_id: (String) the auth id of a rider
start_date: (String) date to start a query from, must be in this format, Y-M-D
end_date: (String) date to end query at must be in this format, Y-M-D
trip_status: (String) the status of the trip. must be either, completed, on_ride, cancel, waiting, on_pickup
page: (String) the desired page to retrieve data
trip_class_complete: (String) if querying for class complete. Must be either yes or no
trip_class: (String) the class of ride. Must be either A, B, C, or D
item_per_page: (String) Items to be served in array as output result. must not be more than 200. Default: 50

Example Request

http://base-url/trip/getTrips?driver_auth_id=9da8a33b-d280-4d1c-b3f0-2848c8ebea90&rider_auth_id=9da8a33b-d280-4d1c-b3f0-2848c8ebea90&start_date=2020-09-03&end_date=2020-09-03&trip_status=completed&page=1&trip_class_complete=yes&trip_class=A
